# Contributing

Thank you for considering contributing to this Next.js Starter Template!

## How to Contribute

1. **Fork the repository** and create your branch from `main` or the latest stable branch.
2. **Install dependencies** and make sure your changes pass all tests and lint checks.
3. **Write clear, concise commit messages.**
4. **Add or update tests** for your changes if applicable.
5. **Open a pull request** with a clear description of your changes and why they are needed.

## Code Style

- This project uses ESLint and Prettier for code quality and formatting.
- Please run `npm run lint` before submitting your PR.
- Keep code modular and colocate tests in `__tests__` folders next to the code they verify.

## Reporting Issues

If you find a bug or have a feature request, please open an issue with as much detail as possible.
